## Лексер
На вход программе принимается исходник с текстом. Текст парсится на допустимые лексемы и создает соответствующие токены.

Для запуска программы требуется:
- Зайти в папку lab1.
- Открыть командную строку и набрать
```
dotnet build
dotnet run ////
```
